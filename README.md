# 🔐 Cybersecurity Awareness Bot

A conversational C# console application that educates users about cybersecurity best practices including password safety, phishing detection, and safe browsing habits.

## ✨ Features

| Feature | Description | Status |
|---------|-------------|--------|
| 🎵 **Voice Greeting** | Plays a recorded welcome message when the application launches | ✅ |
| 🎨 **ASCII Art Logo** | Displays a cybersecurity-themed logo as a header | ✅ |
| 👤 **Personalized Interaction** | Asks for user's name and personalizes all responses | ✅ |
| 💬 **Cybersecurity Q&A** | Answers questions about passwords, phishing, and safe browsing | ✅ |
| 🛡️ **Input Validation** | Handles empty inputs and unrecognized queries gracefully | ✅ |

## 📋 Topics Covered

- **Password Safety** - Strong passwords, passphrases, 2FA, password managers
- **Phishing Detection** - Warning signs, suspicious emails, scam identification
- **Safe Browsing** - HTTPS, secure connections, VPNs, public Wi-Fi risks
